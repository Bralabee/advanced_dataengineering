# dbt Service

This directory is for your dbt project.

## Usage
- Place your dbt models in `dbt/models/`.
- The example model is `models/example.sql`.
- The dbt project config is in `dbt_project.yml`.
- To run dbt:
  ```bash
  docker-compose run --rm dbt run
  ```

See the main README.md for more details.
